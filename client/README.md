# AI Quiz Generator - Frontend

A modern, interactive quiz application powered by AI. This frontend application allows users to generate quizzes from text content instantly, featuring a responsive design and smooth animations.

## Features
- **AI-Powered Generation**: Instantly creating quizzes from provided text.
- **Interactive UI**: Smooth transitions and effects using Framer Motion.
- **Responsive Design**: Built with Tailwind CSS for mobile and desktop compatibility.

## Tech Stack
- **Framework**: React (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Setup Instructions

1. **Prerequisites**: Ensure you have Node.js installed.
2. **Install Dependencies**:
   ```bash
   npm install
   ```
3. **Environment Setup**:
   - Create a `.env` file if necessary (see `.env.example`).

## How to Run

Start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173`.
